# External Skills Catalog

외부 스킬을 링크로만 정리하는 참고 카탈로그입니다.

제3자 소스 코드는 이 저장소에서 재배포하거나 유지보수하지 않습니다. 설치, 업데이트, 라이선스, 보안은 각 upstream 저장소의 정책을 따릅니다.

## 등록 원칙

외부 스킬은 사용자가 하나씩 선택한 뒤 추가합니다. 각 항목에는 다음 정보를 기록합니다.

- 사용 목적
- upstream 저장소
- 공식 설치 명령
- upstream 라이선스 링크
- 마지막 검토 커밋과 날짜
- 설치 부작용 또는 신뢰 관련 주의사항
